# DBU Tech Library System - User Manual

## Quick Start Guide

### 1. Login
- Username: `admin`
- Password: `123`

### 2. Main Modules
- **Member Tab**: Manage library members
- **Book Tab**: Manage book inventory  
- **Catalog Tab**: Extended book information
- **Borrow Tab**: Handle book loans and returns
- **Staff Tab**: Staff management
- **Publisher Tab**: Publisher database
- **Notification Tab**: System notifications

### 3. Key Features
- Real-time data validation
- Automated book availability tracking
- Comprehensive reporting
- Multi-tab interface for easy navigation

## System Requirements
- Python 3.8+
- Tkinter library
- 500MB free space
- 2GB RAM minimum
